# 🚗 Traffic Accident Data Analysis (Task-05)

## 📌 Overview
This project analyzes traffic accident data to identify patterns related to road conditions, weather, and time of day. It also visualizes accident hotspots using maps.

---

## 🎯 Objectives
- Analyze accident trends based on time and weather
- Identify accident-prone locations (hotspots)
- Visualize data using graphs and maps
- Generate meaningful insights from accident data

---

## 📂 Dataset
Dataset used:
https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium (for maps)

---

## ⚙️ Workflow

### 1. Data Loading
- Load dataset using pandas

### 2. Data Cleaning
- Handle missing values
- Convert date/time column
- Extract hour from timestamp

### 3. Data Analysis
- Analyze accidents based on:
  - Time of day
  - Weather conditions

### 4. Visualization
- 📊 Accidents by Hour (Bar Chart)
- 🌦️ Weather Condition Analysis
- 🗺️ Accident Hotspots Map (Folium)
- 🔥 Heatmap for accident density

---

## 📊 Results
- Accidents are higher during peak hours (morning & evening)
- Bad weather conditions increase accident frequency
- Certain regions show high accident density (hotspots)
